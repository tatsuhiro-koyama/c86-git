# Getting start

Gitそのものは、下記のURLよりダウンロードできます。

```
http://git-scm.com/downloads
```

## バージョン管理の開始
- `git config`
- `git init`
  - 今いるディレクトリ以下のファイルをgit管理下におくコマンド

# Record history

- `git add`
  - 変更をステージングにあげる
- `git commit`
  - ステージングにあげた変更をリポジトリに反映する
- `git reset`
  - 変更を取り消す
- `git log`


# Branch
- `git branch`
- `git checkout`
- `git merge`

# Remote
- `git remote`
- `git clone`
  - すでにあるリポジトリを手元に持ってくるコマンド。
- `git fetch`
- `git pull`
- `git push`
